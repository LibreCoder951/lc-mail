LC Mail
--------------------------------------------------------------------------------
This is a simple SMTP client, written in C#.

How to use:
1. Download from release tab.
2. Write your mail into a text file
3. Execute

How to build:
1. Download .NET
2. Download Git
3. Add path to .NET binaries to the path environment variable
4. Open a terminal
5. Type in: git clone https://github.com/LibreCoder951/lc-mail
6. Type in: cd lc-mail
7. Type in: csc *.cs /out:lc-mail(.exe (if you're on windows))
8. Have fun
